Privacy

Linki save bookmarks only on your mechine, Linki will never upload them to server for analysising or tracking.

Quick to add

Linki supports save bookmarks of which you are currently browsing. Currently supports Google Chrome and Safari.

Quick to find

Find bookmarks by keyword or tag name.

Keyboard shortcut support

You can define shortcut to launch find and add window.

Exports JSON / Imports JSON (Plan)

Exports all your data to json file when you want. For now all your data is store here: ~/Library/Application Support/com.liaa.linki/bookmark.db.

iCloud Sync (Plan)

Linki plans to use Apple iCloud Sync to make your bookmarks aviliable across your device.
